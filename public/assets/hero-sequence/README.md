# Hero Image Sequence

This folder contains the image sequence for the scroll-based hero animation.

## Naming Convention

All frames MUST follow this naming convention to ensure the animation logic correctly sequences them:

`frame_XXXX.webp`

Where `XXXX` is a 4-digit zero-padded number starting from `0001`.

**Examples**:
- `frame_0001.webp`
- `frame_0002.webp`
- ...
- `frame_0150.webp`

Ensure all images are optimized `.webp` files for performant loading.
